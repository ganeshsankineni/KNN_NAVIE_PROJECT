**KNN & Naive Bayes Classification Project**

This project demonstrates machine learning model deployment using **K-Nearest Neighbors (KNN)** and **Naive Bayes**. The application predicts the species of Iris flower based on input features and displays performance metrics.

---

**Project Overview**

This is a **Flask-based web application** where users can:
- Input flower measurements  
- Choose a model (KNN / Naive Bayes)  
- Get predictions instantly  
- View accuracy, classification report, and confusion matrix  

---

**Problem Statement**

To classify Iris flower species using:
- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width  

---

**Tech Stack**

- Python  
- Flask  
- NumPy  
- Pandas  
- Scikit-learn  

---

**Model Details**

**KNN (K-Nearest Neighbors)**  
Distance-based algorithm  

**Naive Bayes**  
Probabilistic algorithm based on Bayes theorem  

---

**Performance Metrics**

- Accuracy Score  
- Classification Report  
- Confusion Matrix  

---

**Deployment**

Flask web application  
Can be deployed using **Gunicorn**
